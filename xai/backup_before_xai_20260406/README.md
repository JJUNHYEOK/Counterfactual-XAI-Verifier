# 입법 이슈 탐지 엔진

검색창으로 법령을 찾고, 선택한 법령들 사이의 `중복/충돌/헌법 리스크`를 분석하는 웹 엔진입니다.

## 핵심 기능

- 법령 검색: 국가법령정보 Open API(OC 키)로 법령 검색
- 선택 분석: 대상 법령 1개 + 비교 법령 N개를 선택해 즉시 분석
- 점수 + 이유: 단순 점수뿐 아니라
  - 왜 괜찮은지
  - 왜 보완이 필요한지
  를 법적 근거(헌법 원칙/충돌 조문) 기반 문장으로 출력

## 실행 방법

1. 국가법령정보 Open API 키(OC)를 준비
2. 서버 실행

```powershell
node .\web\server.js
```

3. 브라우저 접속

```text
http://localhost:5050
```

## 화면 사용법

1. `Open API 키(OC)` 입력
2. 검색어 입력 후 `검색`
3. 검색 결과에서
   - `대상` 법령 1개 선택
   - `비교` 법령 1개 이상 선택
4. `선택 법령 분석`

## 점수 구조

- overlap_score: 규율 영역 중복도
- conflict_score: 규범 충돌도(의무/금지/허용/면제)
- constitutional_risk_score: 헌법 원칙 위험도
  - 명확성
  - 비례성(헌법 제37조 제2항)
  - 포괄위임금지(헌법 제75조)
  - 소급입법 제한(헌법 제13조 제2항)
  - 평등원칙(헌법 제11조 제1항)
- total_risk_score = `0.35*overlap + 0.35*conflict + 0.30*constitutional`

## API 엔드포인트

- `GET /api/search-laws?q=검색어&oc=API키`
- `POST /api/analyze-selected`

요청 예시:

```json
{
  "oc": "YOUR_OC_KEY",
  "target": { "id": "000000", "name": "대상 법령" },
  "comparisons": [
    { "id": "111111", "name": "비교 법령 A" },
    { "id": "222222", "name": "비교 법령 B" }
  ]
}
```

## 한계
